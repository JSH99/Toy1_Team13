## 커밋 컨벤션

| 목적     | 설명                                                     |
| -------- | -------------------------------------------------------- |
| feat     | 새로운 기능을 추가할 경우                                |
| fix      | 버그를 고친 경우                                         |
| style    | 코드 포맷 변경, 세미콜론, 코드 수정이 없는 경우          |
| refactor | 프로덕션 코드 리팩토링                                   |
| comment  | 주석 추가 및 변경                                        |
| docs     | 문서를 수정한 경우                                       |
| test     | 테스트 코드 추가, 테스트 리팩토링 (프로덕션 코드 변경 X) |
| chore    | 빌드 테스트 업데이트, 패키지 매니저 변경, 기타 수정      |
| rename   | 파일, 폴더명을 수정하거나 옮기는 작업                    |
| delete   | 파일을 삭제하는 경우                                     |
| build    | 빌드 관련 커밋                                           |
